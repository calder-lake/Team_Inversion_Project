Introduction to the Project:

The final analysis shows data on trauma injuries by age, injury type, and shock index. We were interested in identifying demographic trends for these groups and to assess if shock index is related to age in any meaningful way. The injury types of interest in the final analysis include falls, strikes, and firearms.

Data cleaning
Notes relevant to the final data cleaning and to interpretation of this repository:
Please see the comments related to the commits of files to understand whether it was used in preliminary or later analysis
The codes that use data from the TriNetX_EMER_ICD10.csv file are for the final analysis on the cleaned data
The codes that use data from other spreadshets were for other preliminary analysis. For example, it was determined that we had selected codes and patients of interest incorrectly when sampling. This meant that codes were being misinterpreted due to mismatching meanings between ICD9 and ICD10. We opted to select only data with ICD10 codes in the final analysis. Furthermore, our initial wish to look at motor vehicle accidents was discarded as no appropriate codes could be identified in the data.
